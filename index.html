Pivotal Cards

This is a fork of another project, but made to suit US page sizes rather than ISO page sizes. All the measurements in the CSS are still in mm, but I've just made it so it fills a US letter sized page better. I also changed the cards so that only part of the feature cards gets the background color, to save a little on ink.

To use this, drag the pivotal-cards bookmarklet to your toolbar, then run the bookmarklet while viewing a Pivotal Tracker page. The page will reformat, suitable for printing. Print the pages in Landscape mode, double-sided, and then cut them up. Easy Peasy!
